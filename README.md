# api-snapchat

*Snapchat-like app API created with NodeJS (Express and MongoDB)*

## Getting started

Clone the repo and install dependencies, launch the server

### Clone the repo

```
git clone https://github.com/antoineneff/api-snapchat.git
cd api-snapchat
```

### Install dependencies

```
npm install
```

### Launch the server

```
node server.js
```
